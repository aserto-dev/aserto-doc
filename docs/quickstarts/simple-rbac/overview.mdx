# Simple RBAC Quickstart

The Simple RBAC Quickstart includes
* The **Citadel Demo IDP**, which contains five users based on the Rick & Morty cartoon and several sample .
* An API using the simple-rbac policy, which is implemented in several languages

As you go through the Quickstart, you'll learn the following:
* How to install the Simple RBAC template
* How to browse the directory and examine the manifest
* How to evaluate simple-rbac policy decisions within the Aserto Evaluator
* How to download and run the Simple RBAC backend.
* How to call the backend in order to test out 

## Prerequisites
To follow this Quickstart you'll need to have an Aserto account. If you do not have one, you can create one [here](https://www.aserto.com/#signup). Once you have created your tenant, you can continue.
