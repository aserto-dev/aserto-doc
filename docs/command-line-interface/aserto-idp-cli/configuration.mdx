# Configuration

To configure the CLI plugins, create a YAML file with the following contents (including the information for the plugins you intend to use):

```
logging:
  log_level: LEVEL
plugins:
  auth0:
    auth0_domain:  DOMAIN
    auth0_client_id: ID
    auth0_client_secret: SECRET
  json:
    json_from_file: PATH_TO_FILE
    json_to_file: PATH_TO_OUTPUT_FILE
  aserto:
    aserto_tenant: TENANT
    aserto_authorizer: AUTHORIZER
    aserto_api_key: API_KEY
  okta:
    okta_domain: OKTA_DOMAIN
    okta_api_token: TOKEN
```
