ObjectType represents the kinds of objects that can exist in the directory.

```graphql
type ObjectType {
  id: TypeID!
  isSubject: Boolean!
  name: String!
  displayName: String!
  ordinal: Int!
  status: [Flag!]!
  lifecycle: Lifecycle!
  relationTypes(
  first: Int!
  after: Cursor
): RelationTypeConnection!
}
```


### Fields

#### [`id`](#) ([`TypeID!`](/schema/scalars/type-id))

Unique identifier for the object type.


#### [`isSubject`](#) ([`Boolean!`](/schema/scalars/boolean))

#### [`name`](#) ([`String!`](/schema/scalars/string))

Name of the object type.


#### [`displayName`](#) ([`String!`](/schema/scalars/string))

#### [`ordinal`](#) ([`Int!`](/schema/scalars/int))

#### [`status`](#) ([`[Flag!]!`](/schema/enums/flag))

#### [`lifecycle`](#) ([`Lifecycle!`](/schema/objects/lifecycle))
