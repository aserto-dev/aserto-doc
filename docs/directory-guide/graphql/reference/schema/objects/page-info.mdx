PageInfo contains pagination data about a returned page of results.

```graphql
type PageInfo {
  hasNextPage: Boolean!
  endCursor: Cursor
}
```


### Fields

#### [`hasNextPage`](#) ([`Boolean!`](/schema/scalars/boolean))

True if there are more pages to fetch, false otherwise.


#### [`endCursor`](#) ([`Cursor`](/schema/scalars/cursor))

The cursor to use to fetch the next page.
