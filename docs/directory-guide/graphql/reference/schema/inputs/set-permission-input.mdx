Input to the setPermission mutation.

```graphql
input SetPermissionInput {
  id: ID
  permission: String
  displayName: String
}
```


### Fields

#### [`id`](#) ([`ID`](/docs/directory-guide/graphql/reference/schema/scalars/id))

The ID of an existing permission to update. If provided, a permission with the specified ID must exist.


#### [`permission`](#) ([`String`](/docs/directory-guide/graphql/reference/schema/scalars/string))

The unique permission value.


#### [`displayName`](#) ([`String`](/docs/directory-guide/graphql/reference/schema/scalars/string))

Friendly name for the permission.
