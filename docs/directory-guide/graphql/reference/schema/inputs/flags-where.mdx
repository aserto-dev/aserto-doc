A comparison predicate for filtering on type status fields.

```graphql
input FlagsWhere {
  equals: [Flag!]
  contains: [Flag!]
  notContains: [Flag!]
}
```


### Fields

#### [`equals`](#) ([`[Flag!]`](/docs/directory-guide/graphql/reference/schema/enums/flag))

The type's status is equal to the given flags.


#### [`contains`](#) ([`[Flag!]`](/docs/directory-guide/graphql/reference/schema/enums/flag))

The type's status contains all of the given flags.


#### [`notContains`](#) ([`[Flag!]`](/docs/directory-guide/graphql/reference/schema/enums/flag))

The type's status does not contain any of the given flags.
