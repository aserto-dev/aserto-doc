Input that uniquely identifies a permission.

The fields are mutually exclusive. Exactly one value must be set.

```graphql
input PermissionSelector {
  id: ID
  permission: String
}
```


### Fields

#### [`id`](#) ([`ID`](/docs/directory-guide/graphql/reference/schema/scalars/id))

The permission ID.


#### [`permission`](#) ([`String`](/docs/directory-guide/graphql/reference/schema/scalars/string))

The unique permission value.
