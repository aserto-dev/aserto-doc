## Schema

The [`ObjectType`](schema/objects/object-type) type represent directory object types.

## Listing Object Types

Use the [`objectTypes`](schema/queries/object-types) query to search for object types.

### List All

The following query is used to [paginate](overview#pagination) over all object types:

```graphql
{
  objectTypes(first: 3) {
    nodes {
      id
      name
      isSubject
      displayName
      ordinal
      status
      lifecycle {
        createdAt
      }
    }
    pageInfo {
      hasNextPage
      endCursor
    }
  }
}
```

### Filtering

The [`objectTypes`](schema/queries/object-types) query can also be used to [filter](overview#filters) object types by
providing `where` criteria. The following query selects object types that are subjects and their name contains the
substring `"user"`:

```graphql
{
  objectTypes(
    first: 3,
    where: {isSubject: true, name: {contains: "user"}}
  ) {
    nodes {
      id
      name
      isSubject
      displayName
      ordinal
      status
    }
  }
}
```
