# Existing policy can't be added

## Problem

When attempting to add an existing policy from Github, I get the error "E10010 there was an error interacting with the source code provider: E10022 repo has already been connected to a policy".

![error-repo-already-connected](/troubleshooting/error-repo-already-connected.png)

## Solution

When adding a policy from an existing repository, you need to enable the "Overwrite Aserto push secret" option as shown below:

![check-overwrite-push-secret](/troubleshooting/check-overwrite-push-secret.png)
