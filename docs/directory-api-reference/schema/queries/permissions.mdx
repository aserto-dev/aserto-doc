Enumerate permissions.

```graphql
permissions(
  first: Int!
  after: Cursor
): PermissionConnection!
```


### Arguments

#### [`first`](#) ([`Int!`](/docs/directory-api-reference/schema/scalars/int))

[Required] The maximum number of permissions to return.


#### [`after`](#) ([`Cursor`](/docs/directory-api-reference/schema/scalars/cursor))

[Optional] Pagination cursor for forward pagination.


### Type

#### [`PermissionConnection`](/docs/directory-api-reference/schema/objects/permission-connection)

A paginated collection of Permission instances.
