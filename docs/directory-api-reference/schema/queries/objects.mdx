Find objects that match the given criteria.

```graphql
objects(
  where: ObjectsWhere
  first: Int!
  after: Cursor
): ObjectConnection!
```


### Arguments

#### [`where`](#) ([`ObjectsWhere`](/docs/directory-api-reference/schema/inputs/objects-where))

[Optional] Filtering criteria.


#### [`first`](#) ([`Int!`](/docs/directory-api-reference/schema/scalars/int))

[Required] The maximum number of objects to return.


#### [`after`](#) ([`Cursor`](/docs/directory-api-reference/schema/scalars/cursor))

[Optional] Pagination cursor for forward pagination.


### Type

#### [`ObjectConnection`](/docs/directory-api-reference/schema/objects/object-connection)
