Select a single object type.

```graphql
objectType(
  selector: ObjectTypeSelector!
): ObjectType!
```


### Arguments

#### [`selector`](#) ([`ObjectTypeSelector!`](/docs/directory-api-reference/schema/inputs/object-type-selector))

[Required] Selection criteria. Object types can be selected based on their `id` or `name` field.


### Type

#### [`ObjectType`](/docs/directory-api-reference/schema/objects/object-type)

ObjectType represents the kinds of objects that can exist in the directory.
