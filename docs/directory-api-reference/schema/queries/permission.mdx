Select a single permission.

```graphql
permission(
  selector: PermissionSelector!
): Permission!
```


### Arguments

#### [`selector`](#) ([`PermissionSelector!`](/docs/directory-api-reference/schema/inputs/permission-selector))

[Required] Selection criteria. Permissions can be selected based on their `id` or `permission` field.


### Type

#### [`Permission`](/docs/directory-api-reference/schema/objects/permission)

An action that can be performed on a resource.
