An action that can be performed on a resource.

```graphql
type Permission {
  id: ID!
  permission: String!
  displayName: String
  hash: Hash!
  lifecycle: Lifecycle!
  relationTypes(
  first: Int!
  after: Cursor
): RelationTypeConnection!
}
```
