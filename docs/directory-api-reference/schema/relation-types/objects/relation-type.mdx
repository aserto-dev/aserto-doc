A kind of relation that can exist between subjects and resources.

```graphql
type RelationType {
  id: TypeID!
  objectType: ObjectType!
  name: String!
  displayName: String!
  ordinal: Int!
  status: [Flag!]!
  unions: [RelationType!]!
  hash: Hash!
  lifecycle: Lifecycle!
  permissions(
  first: Int!
  after: Cursor
): PermissionConnection!
}
```
