A comparison predicate for filtering relation types.

All fields are optional. Only relation types that match all given fields are included.

```graphql
input RelationTypesWhere {
  id: TypeIDWhere
  objectType: ObjectTypesWhere
  name: StringWhere
  displayName: StringWhere
  unions: UnionsWhere
  ordinal: IntWhere
  status: FlagsWhere
}
```


### Fields

#### [`id`](#) ([`TypeIDWhere`](/docs/directory-api-reference/schema/inputs/type-idwhere))

Filter relation types by id.


#### [`objectType`](#) ([`ObjectTypesWhere`](/docs/directory-api-reference/schema/inputs/object-types-where))

Filter relation types by their associated object type.


#### [`name`](#) ([`StringWhere`](/docs/directory-api-reference/schema/inputs/string-where))

Filter relation types by name.


#### [`displayName`](#) ([`StringWhere`](/docs/directory-api-reference/schema/inputs/string-where))

Filter relation types by display name.


#### [`unions`](#) ([`UnionsWhere`](/docs/directory-api-reference/schema/inputs/unions-where))

Filter relation types by the relation types they are a union of.


#### [`ordinal`](#) ([`IntWhere`](/docs/directory-api-reference/schema/inputs/int-where))

Filter relation types by sort order.


#### [`status`](#) ([`FlagsWhere`](/docs/directory-api-reference/schema/inputs/flags-where))

Filter relation types by their status flags.
