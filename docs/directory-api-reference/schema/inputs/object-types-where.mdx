A comparison predicate for filtering on object types.

All fields are optional. Only object types that match all given fields are included.

```graphql
input ObjectTypesWhere {
  id: TypeIDWhere
  isSubject: Boolean
  name: StringWhere
  displayName: StringWhere
  ordinal: IntWhere
  status: FlagsWhere
}
```


### Fields

#### [`id`](#) ([`TypeIDWhere`](/docs/directory-api-reference/schema/inputs/type-idwhere))

Filter types by their id.


#### [`isSubject`](#) ([`Boolean`](/docs/directory-api-reference/schema/scalars/boolean))

If true, only include subject types. If false, only include non-subject types.


#### [`name`](#) ([`StringWhere`](/docs/directory-api-reference/schema/inputs/string-where))

Filter types by their name.


#### [`displayName`](#) ([`StringWhere`](/docs/directory-api-reference/schema/inputs/string-where))

Filter types by their display name.


#### [`ordinal`](#) ([`IntWhere`](/docs/directory-api-reference/schema/inputs/int-where))

Filter types by their sort order.


#### [`status`](#) ([`FlagsWhere`](/docs/directory-api-reference/schema/inputs/flags-where))

Filter types by their status.
