A comparison predicate for filtering types on their id field.

```graphql
input TypeIDWhere {
  equals: TypeID
  notEquals: TypeID
  in: [TypeID!]
  notIn: [TypeID!]
}
```


### Fields

#### [`equals`](#) ([`TypeID`](/docs/directory-api-reference/schema/scalars/type-id))

The type's id is equal to the given id.


#### [`notEquals`](#) ([`TypeID`](/docs/directory-api-reference/schema/scalars/type-id))

The type's id is not equal to the given id.


#### [`in`](#) ([`[TypeID!]`](/docs/directory-api-reference/schema/scalars/type-id))

The type's id is equal to one of the given ids.


#### [`notIn`](#) ([`[TypeID!]`](/docs/directory-api-reference/schema/scalars/type-id))

The type's id is not equal to any of the given ids.
