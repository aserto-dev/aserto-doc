Find relations that match the given criteria.

```graphql
relations(
  where: RelationsWhere
  first: Int!
  after: Cursor
): RelationConnection!
```


### Arguments

#### [`where`](#) ([`RelationsWhere`](/docs/directory-api-reference/schema/relations/inputs/relations-where))

[Optional] Filtering criteria.


#### [`first`](#) ([`Int!`](/docs/directory-api-reference/schema/common/scalars/int))

[Required] The maximum number of relations to return.


#### [`after`](#) ([`Cursor`](/docs/directory-api-reference/schema/common/scalars/cursor))

[Optional] Pagination cursor for forward pagination.


### Type

#### [`RelationConnection`](/docs/directory-api-reference/schema/relations/objects/relation-connection)

A paginated collection of relations.
