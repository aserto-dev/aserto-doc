A comparison predicate for filtering on numeric fields.

At most one field may be specified.

```graphql
input IntWhere {
  equals: Int
  notEquals: Int
  in: [Int!]
  notIn: [Int!]
  lessThan: Int
  lessThanOrEqual: Int
  greaterThan: Int
  greaterThanOrEqual: Int
}
```


### Fields

#### [`equals`](#) ([`Int`](/docs/directory-api-reference/schema/common/scalars/int))

The field being compared is equal to the given value.


#### [`notEquals`](#) ([`Int`](/docs/directory-api-reference/schema/common/scalars/int))

The field being compared is not equal to the given value.


#### [`in`](#) ([`[Int!]`](/docs/directory-api-reference/schema/common/scalars/int))

The field being compared is equal to one of the given values..


#### [`notIn`](#) ([`[Int!]`](/docs/directory-api-reference/schema/common/scalars/int))

The field being compared is not equal to any of the given values.


#### [`lessThan`](#) ([`Int`](/docs/directory-api-reference/schema/common/scalars/int))

The field being compared is less than the given value.


#### [`lessThanOrEqual`](#) ([`Int`](/docs/directory-api-reference/schema/common/scalars/int))

The field being compared is less than or equal to the given value.


#### [`greaterThan`](#) ([`Int`](/docs/directory-api-reference/schema/common/scalars/int))

The field being compared is greater than the given value.


#### [`greaterThanOrEqual`](#) ([`Int`](/docs/directory-api-reference/schema/common/scalars/int))

The field being compared is greater than or equal to the given value.
