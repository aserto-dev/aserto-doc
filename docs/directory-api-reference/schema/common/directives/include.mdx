Directs the executor to include this field or fragment only when the `if` argument is true.

```graphql
directive @include(
  if: Boolean!
)
```


### Arguments

#### [`if`](#) ([`Boolean!`](/docs/directory-api-reference/schema/common/scalars/boolean))

Included when true.
