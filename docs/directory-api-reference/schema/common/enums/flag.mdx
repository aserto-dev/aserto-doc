Modifier flags that can be set of object and relation types.

```graphql
enum Flag {
  HIDDEN
  READONLY
  SYSTEM
}
```


### Values

#### [`HIDDEN`](#)

Do not display the type in the Aserto Console.


#### [`READONLY`](#)

Do not allow the type to be modified in the Aserto Console.


#### [`SYSTEM`](#)

The type is internal and not meant for use by end users.
