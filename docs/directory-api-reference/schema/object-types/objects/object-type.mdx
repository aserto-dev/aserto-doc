ObjectType represents the kinds of objects that can exist in the directory.

```graphql
type ObjectType {
  id: TypeID!
  isSubject: Boolean!
  name: String!
  displayName: String!
  ordinal: Int!
  status: [Flag!]!
  hash: Hash!
  lifecycle: Lifecycle!
  relationTypes(
  first: Int!
  after: Cursor
): RelationTypeConnection!
}
```
