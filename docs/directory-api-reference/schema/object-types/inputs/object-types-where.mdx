A comparison predicate for filtering object types.

All fields are optional. Only object types that match all given fields are included.

```graphql
input ObjectTypesWhere {
  id: TypeIDWhere
  isSubject: Boolean
  name: StringWhere
  displayName: StringWhere
  ordinal: IntWhere
  status: FlagsWhere
}
```
