The side of a relation in which an object may be found.

```graphql
enum RelationSide {
  SOURCE
  TARGET
}
```


### Values

#### [`SOURCE`](#)

Object must be on the source side of the relation.


#### [`TARGET`](#)

Object must be on the target side of the relation.
