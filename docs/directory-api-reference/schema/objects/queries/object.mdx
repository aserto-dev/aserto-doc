Select a single object.

```graphql
object(
  selector: ObjectSelector!
): Object!
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>object.<b>selector</b></code>](#)<Bullet />[`ObjectSelector!`](/docs/directory-api-reference/schema/objects/inputs/object-selector) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="input"/> <Badge class="secondary" text="objects"/>
> [Required] Selection criteria. Objects can be selected based on their `id` or the combination of `key` and `type`.
> 

### Type

#### [`Object`](/docs/directory-api-reference/schema/objects/objects/object) <Badge class="secondary" text="object"/> <Badge class="secondary" text="objects"/>
> Objects are the primary nodes in the directory graph.

They be associated with each other using Relations.
