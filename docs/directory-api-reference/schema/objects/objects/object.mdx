Objects are the primary nodes in the directory graph.

They be associated with each other using Relations.

```graphql
type Object {
  id: ID!
  type: ObjectType!
  key: String!
  displayName: String!
  properties: Map!
  hash: Hash!
  lifecycle: Lifecycle!
  relations(
  side: RelationSide!
  options: RelationOptions
  first: Int!
  after: Cursor
): RelationConnection!
}
```
