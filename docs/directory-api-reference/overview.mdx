# Overview

Using Aserto's Directory API you can access the manifest, retrieve and update objects in the directory and work with relations.

## Checks
* Check Permission - Check if a subject has permission on the specified object.
* Check Relation - Check it a subject has requrested relation on the specific object.

## Manifest
* Get Manifest - Retrieve the manifest
* Set Manifest - Set the manifest
* Delete Manifest - Delete the manifest

## Objects
* Retrieve an object by type and id
* Delete an object by type and id
* List all objects of a specific type

## Relations
* Get a relation 
* Set a relation
* List relations
* Delete a relation
