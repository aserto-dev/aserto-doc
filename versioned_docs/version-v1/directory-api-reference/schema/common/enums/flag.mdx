Modifier flags that can be set of object and relation types.

```graphql
enum Flag {
  HIDDEN
  READONLY
  SYSTEM
  SHADOW
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>Flag.<b>HIDDEN</b></code>](#) 
> Hidden types are not displayed in the Aserto Console.
> 

#### [<code style={{ fontWeight: 'normal' }}>Flag.<b>READONLY</b></code>](#) 
> Read-only types are not editable in the Aserto Console.
> 

#### [<code style={{ fontWeight: 'normal' }}>Flag.<b>SYSTEM</b></code>](#) 
> Mask a type as internal.
> 

#### [<code style={{ fontWeight: 'normal' }}>Flag.<b>SHADOW</b></code>](#) 
> Shadow objects share the same key with objects of another type.
>
