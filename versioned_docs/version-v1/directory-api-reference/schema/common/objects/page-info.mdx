PageInfo holds information about a returned page of results.

```graphql
type PageInfo {
  hasNextPage: Boolean!
  endCursor: Cursor
  size: Int!
}
```


### Fields

#### [<code style={{ fontWeight: 'normal' }}>PageInfo.<b>hasNextPage</b></code>](#)<Bullet />[`Boolean!`](/docs/v1/directory-api-reference/schema/common/scalars/boolean) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/> <Badge class="secondary" text="common"/>
> True if there are more pages to fetch, false otherwise.
> 

#### [<code style={{ fontWeight: 'normal' }}>PageInfo.<b>endCursor</b></code>](#)<Bullet />[`Cursor`](/docs/v1/directory-api-reference/schema/common/scalars/cursor) <Badge class="secondary" text="scalar"/> <Badge class="secondary" text="common"/>
> The cursor to use to fetch the next page. If there are no more pages, this field is null.

The value of this field is a cursor that can be used with the `after` parameter to fetch the next page of results.
> 

#### [<code style={{ fontWeight: 'normal' }}>PageInfo.<b>size</b></code>](#)<Bullet />[`Int!`](/docs/v1/directory-api-reference/schema/common/scalars/int) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/> <Badge class="secondary" text="common"/>
